# مولّد باترونات الفساتين المحجبة 👗

تطبيق موبايل احترافي لتصميم وطباعة باترونات الفساتين المحجبة بمقاسات مخصصة.

## المميزات الرئيسية ✨

- ✅ **مقاسات شاملة**: أدخل جميع المقاسات اللازمة للفستان
- ✅ **قوالب جاهزة**: استخدم قوالب محضرة مسبقاً (عباية، فستان بسيط، توب وتنورة)
- ✅ **رسم الباترون**: شاهد الباترون على الشاشة مباشرة
- ✅ **تصدير PDF**: اطبع الباترونات بسهولة
- ✅ **حفظ وتحميل**: احفظ باترونك واستخدمه لاحقاً
- ✅ **حساب الخامة**: احسب كمية النسيج المطلوبة تلقائياً
- ✅ **واجهة عربية**: تجربة كاملة باللغة العربية

## المتطلبات 📋

- Flutter 3.0 أو أحدث
- Dart 3.0 أو أحدث
- Android Studio أو VS Code

## التثبيت 🚀

### 1. استنساخ المستودع
```bash
git clone https://github.com/ahmedtalaat115115/dress-pattern-generator.git
cd dress-pattern-generator
```

### 2. تثبيت المكتبات
```bash
flutter pub get
```

### 3. تشغيل التطبيق
```bash

```

## كيفية الاستخدام 📖

### إنشاء باترون جديد
1. اضغط على "إنشاء باترون جديد"
2. أدخل اسم الباترون
3. اختر نوع الفستان من القائمة
4. أو اختر قالب جاهز
5. أدخل المقاسات (بالسنتيمتر)
6. اضغط "إنشاء الباترون"

### عرض الباترونات المحفوظة
1. اضغط على "الباترونات المحفوظة"
2. اختر الباترون المطلوب
3. شاهد تفاصيل الباترون والمقاسات

### تصدير PDF
1. افتح الباترون
2. اضغط على "تصدير PDF"
3. سيتم حفظ الملف في مجلد المستندات

### حساب الخامة
1. افتح الباترون
2. اضغط على "حساب الخامة"
3. اختر نوع القماش
4. أدخل مساحة الباترون
5. ستحصل على كمية النسيج المطلوبة

## المقاسات المطلوبة 📐

البرنامج يتطلب المقاسات التالية (بالسنتيمتر):

| المقياس | الوصف |
|--------|--------|
| الطول الكلي | الطول الكلي من الكتف للأسفل |
| محيط الصدر | محيط الصدر عند أعمق نقطة |
| محيط الخصر | محيط الخصر الطبيعي |
| محيط الأرداف | محيط الأرداف عند أعلى نقطة |
| عرض الكتفين | المسافة بين الكتفين |
| طول الذراع | من الكتف إلى معصم اليد |
| محيط الرقبة | محيط الرقبة |
| عمق الصدر | من الكتف إلى نقطة الخصر |
| عرض الظهر | عرض الظهر |
| طول الكم | طول الكم المرغوب |
